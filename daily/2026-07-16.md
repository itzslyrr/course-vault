# Thursday, July 16, 2026 — Week 3, Day 12

## 📥 Captured today
- **Day 12: Ship Something Real.** The week's finish — point the whole system at one real build. The framing note is **the three-channel loop**: ① *command* (you → AI, you chat/research/decide) → ② *action* (the AI acts **only** through the one governed doorway) → ③ *perception* (you run it and gate it on your surface, which sets up the next command). Autonomy grows by widening ②; the system stays human by strengthening ③. → [[the-three-channel-loop]]
- The vocabulary named in one place: **the surface** (any dual-legible artifact you and the AI both read/write — the load-bearing idea), **the CLI/doorway** (channel ②), **the Hook** (deterministic fail-closed enforcement — guards the path), **the doctor** (write-format check — required fields · known type · real zone), **the Human Gate** (the AI proposes, you ratify — the author never signs off its own work). **Trustworthy autonomy = judgment (Gate) + enforcement (Hook) + one doorway (CLI).** And *vibe-coding* (a real thing built on a governed trail you can audit) vs *vibe-slop* (output nobody can check).
- The method for driving any build: research → record → project → ADR → plan → scope → tasks → build → run → showcase. The vault holds the trail; the AI does the work; you hold the judgment.
