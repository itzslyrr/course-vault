# Missile Info Sources

- **What:** the public, non-sensitive sources behind the missiles project
- **Where:**
  - Wikipedia — per-system articles (specs, history)
  - CSIS Missile Threat — https://missilethreat.csis.org (analyst-grade public profiles)
  - Wikimedia Commons — freely-licensed photos via `en.wikipedia.org/api/rest_v1/page/summary/<Article>` thumbnails
- **Why saved:** for daily feedings on missile/defense topics, cite from these first; they satisfy the non-sensitive-only rule
