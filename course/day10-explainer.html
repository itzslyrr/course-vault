🧱 The Second Law

Bricking up the wall — the doorway becomes the only way in.

The problem it fixes

Day 9's counter was opt-in. Anyone could still slip a note past it.

The doorway checked writes — but only if you used it. The AI (or an editor, or a script) could still write a file directly into the vault and skip every check. That's the hole. Day 10 closes it by giving the Day-6 guard hook a second law.

The two laws (tool-based)

The PreToolUse guard now watches three tools: Bash | Write | Edit.
# Law 1 (Bash)   git commit / git push   → blocked  "committing is your signature"
# Law 2 (Write/Edit) a write into ~/vault/{knowledge,projects,templates,inbox}/  → blocked
                    "notes go through the doorway. Use the vault command."
# tool files (bin/, doctor.sh) and everything else  → allowed (exit 0)
